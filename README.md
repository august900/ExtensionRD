# Extension [RD]

> An Alpha 1.0.16_02 mod made with RetroMCP.

## About

Extension [RD] is a fan-made AlphaVer-inspired project that explores an alternate branch of Minecraft's development.

The project combines ideas inspired by AlphaVer, RubyDung, and early Minecraft development to create a unique experience featuring experimental mechanics, restored concepts, original lore, and new content.

Extension [RD] is not affiliated with Mojang, Microsoft, or the AlphaVer team.

---

## Features

### Current / Planned Features

* 💎 Ruby Ore
* 🛠️ Ruby Tools
* 🛡️ Ruby Armor
* 🖼️ Canvas Block
* 🎙️ Multiplayer Voice Chat Compatibility
* 🌍 New World Generation Features
* 📜 AlphaVer-Inspired Lore
* 🔧 Experimental Gameplay Systems
* 🎮 Multiplayer Support

More features are planned as development progresses.

## Lore

Within the project's universe, Extension [RD] is considered a recovered development branch built from fragments of abandoned RubyDung and Minecraft concepts.

The meaning of "[RD]" is commonly associated with **RubyDung**, although the project's lore may reveal additional interpretations.
